Library Management System (Java Swing)
Introduction

Library Management System is a desktop application developed using Java Swing to help manage library operations such as managing books, readers, and borrowing activities through a graphical user interface.

The system provides an easy-to-use interface that allows librarians to manage books, track borrowing and returning activities, and maintain reader information efficiently.

This project was developed as part of a university coursework project to practice Object-Oriented Programming, Java GUI development, and database management.

Library management systems like this help automate daily library tasks such as managing books, issuing books to users, and tracking returned books.

Features

🔐 Login System – User authentication to access the system

📚 Book Management – Add, update, delete, and search books

👤 Reader Management – Manage library members information

📖 Borrow Books – Issue books to readers

🔄 Return Books – Update book return information

🔍 Search Function – Search books quickly

🖥 User-friendly Interface – Desktop GUI built with Java Swing

Technologies Used

Language: Java

GUI Framework: Java Swing

Database: MySQL / SQL (depending on configuration)

IDE: NetBeans / IntelliJ / Eclipse

Database Connection: JDBC

System Modules
1. Login

Users must log in with their account to access the system.

2. Book Management

Allows administrators to:

Add new books

Update book information

Delete books

Search for books

3. Reader Management

Manage reader information including:

Adding new readers

Editing reader data

Viewing reader list

4. Borrow & Return

Track library transactions:

Issue books to readers

Record returned books

Manage borrowing history
